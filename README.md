# Projet PHP Hello World

Ce petit projet affiche un message "Bonjour le monde !" avec PHP.

## Lancer le projet

1. Assure-toi d'avoir PHP installé.
2. Ouvre un terminal dans ce dossier.
3. Lance le serveur intégré :

```bash
php -S localhost:8000
```

4. Ouvre `http://localhost:8000` dans ton navigateur.
